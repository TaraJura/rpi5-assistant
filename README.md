# rpi5-assistant

A simple speech recognition + OpenAI Chat + TTS assistant.

## Setup

1. **Clone the repo**:
   ```bash
   git clone git@github.com:TaraJura/rpi5-assistant.git
   cd rpi5-assistant

Create & activate virtualenv (optional but recommended):

python3 -m venv venv
source venv/bin/activate

Install dependencies:

pip install -r requirements.txt

Run the assistant:

python3 main.py


Notes

Make sure you have a working microphone for speech recognition.
Update api_key in main.py with your OpenAI key.
gTTS requires an internet connection for generating TTS.
